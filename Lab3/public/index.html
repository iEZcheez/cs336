Answers for the lab 3 exercises:

  • 3.1
      The purpose of Node.js is an open source server environment that allows JavaScript code to be run outside of a browser.\n Express.js is used for Node. It is a web application for node.js (most of the time). It is a module framework for Node that will "listen" for connection requests from clients.
  • 3.2
      The package.json file has "metadata" for the project. It has information for the npm to give identity to the project and handle the project's dependencies.
  • 3.3
